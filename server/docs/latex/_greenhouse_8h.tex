\subsection{Detailed Description}
Query an Arduino for its environmental readings, record the results to a database, and serve the contents of the database to a web application (as a C\-S\-V file).

The url argument must give the absolute U\-R\-L of the Arduino, as set in the garduino.\-ino file (default\-: 192.\-168.\-1.\-191).
