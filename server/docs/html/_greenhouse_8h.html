Detailed Description

Query an Arduino for its environmental readings, record the results to a database, and serve the contents of the database to a web application (as a CSV file).

The url argument must give the absolute URL of the Arduino, as set in the garduino.ino file (default: 192.168.1.191).

Parameters
urla URL giving the location of the Arduino; ex. http://192.168.1.191
Returns
void; infinite loop
